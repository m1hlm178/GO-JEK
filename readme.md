Aplikasi ini Dibuat untuk memasuki tahap interview Go-Jek dengan memmbuat aplikasi visitor dengan fitur print, notifikasi email dan admin.

## Tutorial Use

1. Clone out source to your server or pc.
2. Root Localhost for visitor public and localhost/login is for Admin Panel.
